Pourquoi utiliser l'analyse statique ?

  • Éviter les erreurs en production
  • Obtenir des metrics sur la qualité du code source et suivre son évolution lors des développements
  • Garantir la cohérence du code écrit par rapport aux standards utilisés
  • Connaître la compatibilité du code source avec une version de PHP, d'un framework ...

Mais aussi :

  • Simplifier le refactoring
  • Identifier les points problématiques lors de la reprise de code (legacy)
